Hệ thống giám sát môi trường RTOS & IoT - Nhóm 15

Dự án sử dụng FreeRTOS trên Arduino Uno để quản lý đa nhiệm và ESP32 làm Gateway gửi dữ liệu lên ThingsBoard Cloud qua MQTT.

Thành phần tệp tin:
* code_chinh_arduino.ino: Xử lý đa nhiệm trên Arduino.
* code_esp32_thingsboard.ino: Nhận dữ liệu Serial và đẩy lên Cloud.
* code_website.ino: Giao diện Web Server giám sát tại chỗ.
